Git 快速入门笔记(一)——基础知识

  1. Git是什么？

    Git是目前世界上最先进的分布式版本控制系统。在Git管理的项目中，Git会记录每一次修改，以便操作人员进行文件/源码的版本控制与管理。可以参考一下廖雪峰大佬博客的例子。

    注：本文也是一边看廖雪峰大佬的博客一边实践整理出的个人学习笔记。

    例如，我们需要向learngit库中添加readme.txt文件，只需要：

    • 在learngit文件夹下创建新文件readme.txt

    • 用git add命令将文件添加到本地仓库

    • 用git commit 命令将文件提交到仓库

    Git中，用HEAD表示当前版本，上一个版本就是HEAD^，上上一个版本就是HEAD^^，当然往上100个版本写100个^比较容易数不过来，所以写成HEAD~100。

    若是修改后已经添加到暂存区并且还未提交，则可以使用git reset HEAD <file>将暂存区的修改撤销：

    若是修改后已经添加到暂存区并且提交，则需要使用版本回退。
