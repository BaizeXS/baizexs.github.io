Python入门笔记——time库

Python入门笔记—— time 库的使用

  1. time 库是Python中处理时间的标准库

    • 计算机时间表达
    • 提供获取系统时间并格式化输出功能
    • 提供系统级精确计时功能，用于程序性能分析

  2. time 库的使用

  3. time 库主要包括三类函数

    • 时间获取：time(); ctime(); gmtime();
    • 时间格式化：stwftime(); strptime();
    • 程序计时：sleep(); perf_counter();

  4. 时间获取

函数 功能
time() 获取当前时间戳，即计算机内部的时间值，该时间值为浮点数。
>>>time.time()
1630242266.687503
ctime() 获取当前时间并以易读方式表示，返回字符串。
>>>time.ctime()
‘Sun Aug 29 21:06:42 2021’
gmtime() 获取当前时间并表示为计算机可处理的时间格式。
>>>time.gmtime()
>> time.struct_time(tm_year=2021, tm_mon=8, tm_mday=29, tm_hour=13, tm_min=8, tm_sec=55, tm_wday=6, tm_yday=241, tm_isdst=0)>>> time.struct_time(tm_year=2021, tm_mon=8, tm_mday=29, tm_hour=13, tm_min=8, tm_sec=55, tm_wday=6, tm_yday=241, tm_isdst=0)

  1. 时间格式化

    • 格式化：类似于字符串格式化，需要有展示模板
    • 展示模板由特定的格式化控制符组成
函数 功能
strftime(tpl,ts) tpl是格式化模板字符串，用来定义输出效果
ts是计算机内部时间类型变量
>>>t = time.gmtime()
>>>time.strftime(“%Y-%m-%d %H:%M:%S”,t)
‘2018-01-26 12:55:20’
strptime(str,tpl) str是字符串形式的时间值
tpl是格式化模版字符串，用来定义输入效果
>>>timeStr = ‘2018-01-01 12:00:00’
>>>time.strptime(timeStr, “%Y-%m-%d %H:%M:%S”)

  1. 程序计时

    程序计时应用广泛

    • 程序计时指起止动作所经历时间的过程
    • 测量时间：perf_counter()
    • 产生时间：sleep()
函数 功能
perf_counter() 返回一个CPU级别的精确时间计数值，单位为秒。一般连续使用来获得时间差
注意，第一次perf_counter()是往后所有perf_counter()的基准时间！
>>> start = perf_counter()
>>> end = perf_counter()
end - start
sleep(s) 该函数使得程序休眠 s 秒。
